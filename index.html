UI cheat sheet: Spacing friendships

Diseño Espaciado

Para mí, la forma más fácil de ver la habilidad de un diseñador de interfaces es cómo utiliza el espaciado en su diseño. El color, la tipografía, las imágenes, etc., suelen venir dados en una guía de marca, pero utilizar correctamente el espaciado es una habilidad. Después de un tiempo se convierte en algo natural, pero sólo después de haber aprendido y utilizado las reglas básicas.
